# KinKeep Marketing Website

This is the marketing landing page for the KinKeep application, built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack
- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS (Deep Dark Mode Theme)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design System
- **Theme**: Dark Mode
- **Background**: Zinc-950 with radial gradient
- **Components**: Glassmorphism (White/5 opacity, blur-xl)
- **Typography**: Inter
